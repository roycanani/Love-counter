# Love-counter

A romantic, modern countdown web app to the reunion on **February 25 at 19:20 GMT** (21:20 Israel time), dedicated to Jade.

## Run

Open `index.html` in your browser.

## Background music

Background music is streamed remotely from YouTube using the query "Ordinary Alex Warren".

- It loops in the background.
- If autoplay is blocked, press the music button to start playback.

## Files

- `index.html` – app structure
- `styles.css` – styling
- `script.js` – countdown logic
